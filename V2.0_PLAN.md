## 三、技术方案

### 3.1 技术栈
- React 18 + Ant Design 5
- Recharts (图表库)
- Day.js (日期处理)

### 3.2 组件架构
```
src/
├── components/
│   ├── BacktestResult/
│   │   ├── EquityCurve.js        # 收益曲线
│   │   ├── HoldingChange.js      # 持仓变化
│   │   ├── TradeTimeline.js      # 交易时间轴
│   │   ├── MonthlyReturn.js      # 月度收益
│   │   └── RiskDashboard.js      # 风险仪表盘
│   ├── StrategyConfig/
│   │   ├── StrategyForm.js       # 策略参数表单
│   │   ├── ParameterPresets.js   # 参数预设
│   │   └── Validation.js         # 参数验证
│   └── common/
│       ├── StatCard.js           # 统计卡片
│       └── RiskRadar.js          # 风险雷达图
├── hooks/
│   ├── useBacktest.js            # 回测数据Hook
│   └── useRiskMetrics.js         # 风险指标计算
└── utils/
    ├── riskCalculator.js         # 风险指标计算工具
    └── formatters.js             # 格式化工具
```

### 3.3 API 需求（可能需要后端配合）
- 回测结果增强：`GET /api/v1/backtests/{id}/detail` 返回完整持仓历史
- 风险指标计算：后端计算或前端基于equity_curve计算
