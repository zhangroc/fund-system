## 数据采集实现详情 (2026-03-10)

### 已完成功能:

1. **AkShare 集成** (`app/services/data_collector.py`)
   - 基金列表采集 (`collect_fund_list`)
   - 基金净值采集 (`collect_fund_nav`)
   - 基金详情更新 (`update_fund_details`)
   - 基金持仓采集 (`collect_fund_holdings`)
   - ETF 历史数据 (`collect_etf_hist`)
   - 基金排名数据 (`collect_fund_rank`)
   - 增量净值更新 (`update_latest_nav`)

2. **API 端点** (`app/api/data_collection.py`)
   - POST /api/v1/data/collect/funds - 采集基金列表
   - POST /api/v1/data/collect/nav - 采集基金净值
   - POST /api/v1/data/collect/details - 更新基金详情
   - POST /api/v1/data/collect/holdings/{fund_code} - 采集持仓
   - GET /api/v1/data/collect/etf/{symbol} - ETF历史数据
   - GET /api/v1/data/collect/rank - 基金排名
   - POST /api/v1/data/update/nav - 增量更新净值
   - POST /api/v1/data/scheduled - 定时任务

3. **定时采集脚本** (`scripts/scheduled_collector.py`)
   - 支持 full/nav/details 三种模式
   - 可配置 crontab 定时执行

### 使用方式:

```bash
# 增量更新净值（推荐每天执行）
python scripts/scheduled_collector.py --mode nav

# 完整采集（首次运行）
python scripts/scheduled_collector.py --mode full

# 更新基金详情
python scripts/scheduled_collector.py --mode details
```

### Crontab 配置示例:
```bash
# 每天下午6点增量更新净值
0 18 * * * cd /home/ubuntu/.openclaw/workspace/fund-system/backend && python3 ../scripts/scheduled_collector.py --mode nav >> /var/log/fund_collector.log 2>&1
```
